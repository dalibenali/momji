# momji
1)Create databases with scl files (schema.sql, test_schema.sql)
Create .env file in root directory like :

    DB_HOST= db host
    DB_USER= db user
    DB_PASS= db password
    DB_PORT= db port (3306)
    DB_NAME=momji
    DB_TEST_NAME=test


Install packages : yarn install

Start server : yarn run dev

Run the tests : yarn run test
